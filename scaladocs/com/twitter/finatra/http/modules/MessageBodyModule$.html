MessageBodyModule 

object MessageBodyModule extends TwitterModule

Provided implementations for the com.twitter.finatra.http.marshalling.DefaultMessageBodyReader and the com.twitter.finatra.http.marshalling.DefaultMessageBodyWriter.

Also binds the com.twitter.finatra.http.internal.marshalling.mustache.MustacheMessageBodyWriter which provides the mustache rendering support for the framework.
