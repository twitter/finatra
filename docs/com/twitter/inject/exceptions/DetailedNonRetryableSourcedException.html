  7. val details: Seq[Any]

    Details about this exception suitable for stats.

    Details about this exception suitable for stats. Each element will be converted into a string. Note: Each element must have a bounded set of values (e.g. You can stat the type of a tweet e.g., "protected" or "unprotected", but you shouldn't include an actual tweet id, e.g. "121314151617181912131").

    This is different from message which will be logged when this exception occurs. Typically message contains any unbounded details of the exception that you are not able to stat such as an actual tweet id (see above).
