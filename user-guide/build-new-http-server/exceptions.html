Basics

It is recommended that you use exceptions for flow control in your controller and services and rely on the com.twitter.finatra.http.exceptions.ExceptionMapper to convert exceptions into proper HTTP responses. Finatra provides a default FinatraDefaultExceptionMapper which provides high-level mapping for exceptions. However, you are free to register additional mappers or to override the default mapper altogether.
